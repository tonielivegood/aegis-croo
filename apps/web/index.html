06 / BUILDER CONTRACT

Developer integration

Same-origin JSON API

Call Aegis before your agent acts. Start with health and risk-check; use local/mock order endpoints only when testing integration contracts.

Health & posture

curl http://127.0.0.1:8000/health
curl http://127.0.0.1:8000/cap/status

Risk check

curl -X POST http://127.0.0.1:8000/risk-check \
  -H "Content-Type: application/json" \
  --data @risk-request.json

Mock A2A

curl -X POST http://127.0.0.1:8000/a2a/mock-order \
  -H "Content-Type: application/json" \
  --data @mock-order.json

Local order & proof

curl -X POST http://127.0.0.1:8000/orders -H "Content-Type: application/json" --data @order.json
curl http://127.0.0.1:8000/orders/{order_id}
curl http://127.0.0.1:8000/proof/{proof_id}
